# HelloDrum-2
This is a remix, (partial) rewrite and continued development version of the original HelloDrum project by Ryo Kosaka.
For more details, please check LICENSE.txt, listed above.

## About
HelloDrum-2 is a HelloDrum successor, which is a DIY, low-cost open-source e-drum kit that has near-infinite customization possibilities, all crafted in your own home with easy-to-come-by tools.

Beware though, as this project requires (minor) soldering, a bit of coding, a couple of hours of free time and a drink or two. I'm currently working on making the kit easier to build for all people, so if all above seems complicated, don't worry!
